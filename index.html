Rexive

Rexive is a responsive structure for mobile, tablet, and desktop. The order there is important, and what sets Rexive apart from every other grid system.

Rexive is light to the point that it's ugly. It assumes you have your own style.

Only what you need

The intention of Rexive is to be light, easy, powerful, and out of the way of your styles.

The javascript

Together with minimal css, addresses fixed positioning problems and scrolling on touch devices.

This piece of javascript is completely optional, and if your pages have layered elements set apart by z-index, this scrolling solution isn't optimal for your use.

CSS

#body {
  -webkit-overflow-scrolling: touch;
}

JS (simplified for viewing)

$('#body').css('overflow-y', 'scroll');
$(window).onresize(function(){
  $('#body').css('height', $(window).height() - $('#header').outerHeight(true));
});
